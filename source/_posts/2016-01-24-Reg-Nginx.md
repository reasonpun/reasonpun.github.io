# Nginx 日志配置格式

```
log_format  main
        '[$upstream_addr] $remote_addr [$time_local] "$request" $status '
        '"$request_body" $body_bytes_sent "$http_referer" "$http_user_agent" '
        'RESP:$upstream_response_time '
        'REQ:$request_time';
```

# 正则表达式

## Python版本

```
p = re.compile(
            r"\[\-?[\d.\:]*[\ \,]*?.*?\]\ [\d.\:]*\ \[(\d+)/(\w+)/(\d+)\:(\S+)\ [\S]+\]\ \"(\S+)\ (\S+)\ .*?\"\ (\d+)\ \"(.*?)\"\ (\d+)\ \"([^\"]*)\"\ \".*?\" .*?")
m = re.findall(p, line)
day = m[0][0]
month = m[0][1]
year = m[0][2]
ttime = m[0][3]
method = m[0][4]
request = m[0][5]
status = m[0][6]
```
