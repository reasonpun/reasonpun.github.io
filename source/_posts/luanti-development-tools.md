# 开发工具
这一页整理了**开发 Luanti 游戏/模组、改引擎**必备的实用工具，全是圈内人常用的神器！

<!--more-->

---

## 引擎开发工具
- **Development Test**：引擎功能测试专用游戏
- **/luanti/util**：引擎自带的各种维护脚本

---

## 通用 Lua 工具
- **luacheck**：Lua 代码检查器，帮你抓 bug、规范写法
- **busted**：Lua 单元测试框架，保证代码稳定
- **Lua pattern viewer**：Lua 正则可视化工具（类似 Regexr）
- **Warr1024’s Lua Code Tools**：代码格式化 + 自动生成局部变量

---

## 调试 / 性能分析
- **debug**：Luanti 调试库，比自带 log 强得多
- **LuaJIT Profiler**：模组性能分析器，生成火焰图
- **Warr1024 的 Docker 服务端镜像**：自带调试 + 性能分析工具

---

## 语法高亮 / 自动补全
- **Luanti Tools**：VS Code / 开源编辑器插件，提供 Luanti API 全自动补全

---

## 结构（Schematics）工具
- **Schematic Editor**：游戏内直接导入、编辑、导出结构文件
- **MTS Editor**：独立编辑器，支持 Luanti/Minecraft 多种格式
- **Lua2Mts**：把 .lua 结构转成 .mts 官方格式
- **ASCII Art Schematics**：用 ASCII 字符画管理结构

---

## 群系（Biomes）工具
- **Luanti Biome Point Visualizer**：可视化编辑群系的温度/湿度， Voronoi 图直观展示

---

## 柏林噪声（Perlin Noise）工具
- **Perlin Explorer**：游戏内实时调试各种噪声效果
- **Perlin noise tuner**：网页可视化调试噪声参数

---

## 3D 模型
- **Blender**：做动画模型必备神器
- **Blockbench**：做体素静态模型超好用

---

## 图片压缩优化
- **optipng**：PNG 无损压缩
- **pngcrush**：PNG 极致压缩
- **pngquant**：PNG 有损压缩（体积更小）
- **guetzli**：JPG 高压缩质量工具

---

## UI 编辑（Formspecs）
- **formspec_editor**：游戏内实时写 UI、实时预览
- **Luanti Formspec Editor**：网页版可视化 UI 编辑器

---

## 翻译工具
- **Gettext utils**：新版 .po 格式翻译维护工具
- **mod_translation_updater**：旧版 .tr 翻译文件自动更新脚本

---

## ContentDB 发布
- **cdbrelease**：ContentDB 资源一键发布自动化工具

---

## 杂项神器
- **Definition Ripper**：提取方块/物品定义，生成静态外壳
