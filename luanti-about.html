🎮 关于咱 Luanti（前身为 Minetest）

Luanti 就是免费开源的体素游戏引擎天花板，自带专属分发平台和客户端，玩方块、造世界、开服务器、改引擎全都能搞定！不管你是纯玩的玩家、搞创作的大佬、开服的卷王，还是改引擎的代码大神，在这都能找到入门攻略，少走十年弯路！

🛠️ 网站小科普

这网站是用 Hugo 搭的，皮肤是 Hugo Book 主题，源码全放 GitHub 上了，欢迎各路大神来提交修改，一起完善文档，人人都是贡献者！想参与的直接看「文档贡献指南」就行～

🚀 入门速查（按身份对号入座）

咱这文档直接按人群分类，不用瞎翻找：

  • 纯玩党：教你装客户端、装模组、装游戏，还有基础玩法，小白秒上手

  • 创作大神：模组制作、游戏开发、材质包设计、翻译，各种造东西的教程全在这

  • 服主卷王：多人服务器搭建一条龙，开服不踩坑

  • 引擎大佬：教你怎么给 Luanti 引擎写 C++ 代码，硬核改造随便玩

📜 Luanti 文档使命宣言

咱这文档就是 Luanti 项目的核心百科全书！

所有贡献者和写手一起肝，打造经过实测、通俗易懂、分类清晰、搜啥都有的超全资料，紧跟 Luanti 最新版本。

文档覆盖这些硬核内容：

  • 超全 API 参考、实例、教程
  • 引擎底层结构与参考指南
  • 服务器和平台使用教程（玩家友好版）

这项目 2024 年 12 月才启动，未来规划直接看 GitHub 第 113 号问题：路线图！
